# 🌊 Flood Escape Lab — 화면설계서

**버전**: 1.0.0
**작성일**: 2026-06-29
**기준 문서**: SRS v1.3.2
**대상 사용자**: 초등학교 1~2학년
**비주얼 톤**: 밝고 친근한 교육용 일러스트 스타일 (배경 `#EFF6FF`, 위험 강조색 `#FF3B30`)
**디자인 원칙**: 초등 1~2학년이 공포감 없이 몰입할 수 있도록 밝은 배경 기반 설계. 긴박감은 색상이 아닌 애니메이션·수위 게이지·효과음으로 표현. 노란 우비 캐릭터 + 단면도 일러스트 형식.

### 사용자 행동 → 시스템 반응

| 사용자 행동 | 시스템 반응 |
|-----------|-----------|
| 화면 진입 | 기상청 API 호출 → 시나리오 데이터 로딩 |
| API 성공 | 시나리오 카드에 실제 날짜·강우량 표시 |
| API 실패 | 폴백 시나리오 자동 전환 + 안내 문구 표시 |
| 잠긴 카드 탭 | "이 단계는 아직 잠겨 있어요 🔒" 토스트 |
| 시작하기 탭 | 구조대상 선택 화면으로 전환 |

### 비주얼 톤
- 배경: `#EFF6FF` (하늘 파랑)
- 카드: `#FFFFFF` + 하늘색 테두리 `#BFDBFE`
- 제목: `#1E3A5F` 진한 파랑 굵은 글씨 + 물방울 애니메이션
- 잠긴 카드: opacity 40% 회색
- 전체 분위기: 밝고 친근한 교육용 일러스트. 노란 우비 캐릭터 상단 배치.

## Screen 2. 구조대상 선택

### 화면 목적
누구를 구할지 직접 고른다. 3명을 선택해서 책임감과 몰입감을 높인다.

### 온보딩 문구
> 🏃 **누구를 안전한 곳으로 안내할까요?**
> 3명을 골라요.

### 주요 컴포넌트

| 컴포넌트 | 설명 |
|---------|------|
| 선택 카운터 | "0/3 선택됨" → 탭할수록 증가 |
| 구조대상 카드 | 이모지 + 이름. 탭하면 선택/해제 |
| 확인 버튼 | 3명 선택 완료 시 활성화 → 장비 선택으로 |

### 구조대상 목록

| 이모지 | 대상 |
|-------|------|
| 👩‍🏫 | 선생님 |
| 🧒 | 우리 반 친구들 |
| 🐶 | 반려견 |
| 👨‍👩‍👧‍👦 | 우리가족 |
| 🎀 | 티니핑 |
| 🎤 | 아이브 |

### 선택 카드 상태

```
선택됨: 파란 테두리 + ✅
미선택: 회색 테두리
3명 초과 시도: 카드 흔들림 + "3명까지만 고를 수 있어요!" 토스트
```

### 사용자 행동 → 시스템 반응

| 사용자 행동 | 시스템 반응 |
|-----------|-----------|
| 카드 탭 | 선택 + 카운터 증가 + 파란 테두리 |
| 선택된 카드 다시 탭 | 해제 + 카운터 감소 |
| 4번째 탭 시도 | 흔들림 + 토스트 메시지 |
| 3명 완료 | 확인 버튼 빨간색으로 활성화 |
| 확인 버튼 탭 | 선택 저장 → 장비 선택 화면 |

### 비주얼 톤
- 선택 카드: 파란 테두리 `#1F6FEB` + 배경 살짝 밝아짐
- 확인 버튼: 3명 선택 전 회색, 완료 후 `#FF3B30`

## Screen 3. 장비 선택 (Equipment)

### 화면 목적
방재 장비를 골라서 침수에 대비한다. 예산이 한정되어 있다.

### 온보딩 문구
> 💰 **500원으로 장비를 골라요!**
> 장비에 따라 결과가 완전히 달라져요!

### 주요 컴포넌트

| 컴포넌트 | 설명 | 관련 FR |
|---------|------|---------|
| 예산 바 | 남은 예산 실시간 표시. 0원이면 빨간색 변환 | FR-02, FR-03 |
| 장비 카드 × 4 | 이름 + 이모지 + 가격 + 효과 한 줄 설명 | FR-02 |
| 구매 버튼 | 예산 부족 시 회색 비활성화 | FR-03 |
| 반납 버튼 | 구매한 장비 옆 표시. 탭하면 예산 복구 | FR-04 |
| 장비 없이 시작 | 하단 작은 버튼. 무방비 체험용 | FR-05 |
| 시작 버튼 | "🚨 구조 시작!" | FR-06 |

| 이모지 | 이름 | 가격 | 효과 |
|-------|------|------|------|

### 사용자 행동 → 시스템 반응

| 사용자 행동 | 시스템 반응 |
|-----------|-----------|
| 장비 카드 탭 | 구매 + 예산 바 감소 + 카드 강조 |
| 예산 초과 시도 | 버튼 비활성 + 카드 흔들림 |
| 반납 버튼 탭 | 예산 복구 + 카드 원래대로 |

### 비주얼 톤
- 구매 가능: `#FFFFFF` + 파란 테두리 `#1F6FEB`
- 구매 완료: 초록 체크 + 초록 테두리 `#3FB950`
- 예산 바: 많을수록 파랑 → 적을수록 빨강
- 시작 버튼: `#FF3B30`

### 태풍 시나리오 B·C 전용 — 예보 변화 이벤트

시나리오 B(태풍 근접) 또는 C(태풍 직접) 플레이 중 **10초 시점**에 예보 업데이트 팝업이 등장한다.

```
┌──────────────────────────────────────┐
│ 🌀 태풍 예보 업데이트                 │
│                                      │
│ 처음 예보:                            │
│ 태풍은 바다 쪽으로 지나갈 것 같아요.  │
│                                      │
│ ➡ 새 예보:                           │
│ 태풍이 도시 쪽으로 가까워졌어요!      │
│ 비가 더 많이 올 수 있어요. ⚠️        │
└──────────────────────────────────────┘
```

| 예보 변화 | 게임 영향 |
|---------|---------|
| 태풍이 멀어짐 | 강수량 감소, 침수 속도 느려짐 |
| 태풍이 가까워짐 | 강수량 증가, 침수 속도 빨라짐 |
| 경로가 도시로 이동 | 하천범람 이벤트 발생 (수위 15cm 즉시 상승) |
| 이동 속도 빨라짐 | 남은 시간 체감 감소 (타이머 깜빡임 가속) |
| 이동 속도 느려짐 | 비가 오래 내려 누적 침수 증가 |

**시나리오 B 특수 효과**: 바람 강함 → 🚨 경보시스템 반응이 1초 지연

**시나리오 C 특수 효과**: 하천범람 발생 시 수위 15cm 즉시 상승 이벤트 + "🚨 하천이 넘쳤어요!" 경고 배너

### 태풍 시나리오 학습 포인트 (결과 화면에 표시)

**시나리오 B:**
- ✅ 태풍이 가까이 오면 비와 바람이 함께 강해져요.
- ✅ 지하 공간은 더 빨리 위험해져요.
- ✅ 물막이판 + 경보시스템 조합이 유리해요.

**시나리오 C:**
- ✅ 태풍이 가까이 오면 지하 공간은 매우 위험해요.
- ✅ 물이 차오르기 전에 바로 나가야 해요.
- ✅ 배수펌프 + 경보시스템 + 탈출사다리 조합이 유리해요.
- ✅ 예보를 계속 확인해야 하는 이유가 여기 있어요.

### 비주얼 톤
- 배경: `#EFF6FF` + 물결 파문 애니메이션
- 물 색상: 검푸른 → 수위 높을수록 탁한 갈색
- 화면 가장자리: CRITICAL 시 빨간 빛 번짐
- 텍스트: CRITICAL일 때 흰 글씨 깜빡임
- **태풍 시나리오 전용**: SVG 배경에 바람 줄기 애니메이션, 예보 팝업 카드 (주황 테두리)

## Screen 4-A. 패널티 화면 (실패 시)

### 화면 목적
왜 실패했는지 알려준다. 안전 수칙을 읽어야 다시 도전할 수 있다.

### 화면 구성

```
🚨 긴급 복습 시간!

이번에는 구조가 조금 늦었어요.
다시 도전하기 전에 안전 수칙을 확인해요.

왜 늦었을까요?
• 경보가 늦게 울렸어요
• 물막이판이 없어서 물이 빨리 들어왔어요
• 탈출사다리가 없어서 대피가 늦었어요

━━━━━━━━━━━━━━━━━━━━

📋 안전 수칙 3가지

☐ 1. 지하 공간에 물이 들어오면 바로 나가요.
☐ 2. 물이 무릎까지 오기 전에 대피해요.
☐ 3. 혼자 해결하지 말고 어른에게 알려요.

[안전 수칙 확인했어요 ✅]

💡 다음 도전에서 경보 힌트 1회 제공
```

### 규칙

| 규칙 | 내용 |
|------|------|
| 즉시 재도전 | 불가 |
| 재도전 조건 | 안전 수칙 3개 확인 후 버튼 활성화 |
| 보너스 | 다음 플레이에서 경보 힌트 1회 자동 제공 |

### 사용자 행동 → 시스템 반응

| 사용자 행동 | 시스템 반응 |
|-----------|-----------|
| 안전 수칙 탭 | ☐ → ✅ 체크로 변경 |
| 3개 모두 체크 | "안전 수칙 확인했어요" 버튼 활성화 |
| 버튼 탭 | 힌트 1회 부여 + 장비 선택 화면으로 이동 |

### 비주얼 톤
- 상단 경고 영역: `#FF3B30` 배경 + 흰 글씨
- 안전 수칙 카드: `#FFFFFF` + 주황 테두리
- 확인 버튼: 3개 체크 전 회색, 완료 후 빨간색

## Screen 5. 결과 (Result)

### 화면 목적
얼마나 잘했는지 알려준다. 왜 이런 결과가 나왔는지 배운다.

### 성공 시 — 학습목표 화면

```
🏆 성공!
모두 안전한 곳으로 대피했어요.

━━━━━━━━━━━━━━━━━━━━

오늘 배운 것

✅ 비가 많이 오면 지하 공간은 빨리 위험해져요.
✅ 물이 차오르기 전에 바로 나가야 해요.
✅ 경보와 물막이판은 대피 시간을 벌어줘요.
✅ 혼자 해결하지 말고 어른에게 알려야 해요.

━━━━━━━━━━━━━━━━━━━━

오늘의 안전 약속
"물이 차오르면 기다리지 않고 바로 밖으로 나가요."

[다시 도전]   [다음 단계 →]
```

### 등급 기준

| 등급 | 조건 | 문구 |
|------|------|------|
| S | 선택 대상 전원 탈출 | 🏆 완벽해요! 모두 탈출했어요! |
| A | 필수 전원 + 선택 1명 이상 | 🎉 잘했어요! 거의 다 탈출했어요! |
| B | 필수(선생님+친구들) 전원 탈출 | 👍 괜찮아요! 조금 더 잘할 수 있어요! |
| C | 필수 1명 탈출 실패 | 😰 위험했어요! 다시 해볼까요? |
| F | 필수 전원 탈출 실패 | → 패널티 화면 전환 |

### 결과 상세 패널

```
┌────────────────────────────────┐
│ 👩‍🏫 선생님         ✅ 탈출 성공 │
│ 🧒 우리 반 친구들   ✅ 탈출 성공 │
│ 🐶 반려견           ❌ 미탈출   │
├────────────────────────────────┤
│ 💧 최고 수위: 68cm             │
│ 🚨 경보 울린 시간: 12초        │
└────────────────────────────────┘
```

### 다음 단계 해금 조건

| 현재 레벨 | 조건 | 해금 |
|---------|------|------|
| 1단계 (부산) | B등급 이상 | ⭐⭐ 2단계 (오송) |
| 2단계 (오송) | B등급 이상 | ⭐⭐⭐ 3단계 (강남) |

### 비주얼 톤
- S등급: 금빛 `#FFD700` 글로우
- A등급: 초록 `#3FB950` 글로우
- B등급: 파랑 `#1F6FEB` 글로우
- C등급: 주황 `#FF8C00` 글로우
- F등급: → 패널티 화면으로 전환 (결과 화면 미표시)

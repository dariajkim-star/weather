# 🌊 Flood Escape Lab

https://lovable.dev/projects/5090b4fa-b549-4c60-ab87-7f85c4a23531

> 초등학생 대상 지하 침수 탈출 체험 학습게임

기상청 실제 집중호우 데이터를 기반으로, 구조대상을 선택하고 10초 안에 탈출시키는 침수 시뮬레이션 게임입니다.

**공모전**: 기상청·기후에너지환경부 주관 「기상·기후 AI 해커톤 2026」 출품작

---

## 🎮 게임 화면 흐름

```
로비 → 구조대상 선택 → (장비 선택) → 플레이 → 결과(S/A/B) or 패널티(C/F)
                                              ↓ 1단계 성공 시
                                       🎁 보상 장비 선택
```

---

## 📋 시나리오 구성 (3단계)

| 단계 | 유형 | 설명 | 강도 | 난이도 | 장비 | 해금 |
|------|------|------|------|-------|------|------|
| ⭐ 1단계 | 집중호우 A | 도시 전체에 비가 쏟아져요 (2022 강남) | 141mm/h | 어려움 | ❌ 장비 없음 (필수) | 시작부터 |
| 🌀 2단계 | 태풍 근접 B | 태풍이 도시 근처를 지나가요 | 강함 | 보통 | 🎁 보상 장비 1개 | 1단계 B+ |
| 🌀🌀 3단계 | 태풍 직접 C | 태풍이 도시를 바로 지나가요 | 매우 강함 | 어려움 | 💰 500원 자유 선택 | 2단계 B+ |

> 탈출 시간: **10초** | 데이터 출처: 기상청 ASOS API

## ✨ 핵심 기능

### 1. 1단계 — 장비 없이 체험
장비 선택 화면 없이 바로 플레이. "장비가 없으면 어떻게 되는지" 먼저 체험.
성공 시 🎁 **보상 장비 1개 선택** → 2단계에서 사용.

### 2. 구조대상 선택 (6명 중 3명)
👩‍🏫 선생님 · 🧒 우리 반 친구들 · 🐶 반려견 · 👨‍👩‍👧‍👦 우리가족 · 🎀 티니핑 · 🎤 아이브

### 3. 방재 장비 (2·3단계)

| 장비 | 가격 | 효과 |
|------|------|------|
| 🚧 물막이판 | 200원 | SVG 입구 장벽 등장, 수위 상승 40% 감소 |
| ⚙️ 배수펌프 | 250원 | SVG 하단 펌프, 수위 10cm 즉시 감소 |
| 🚨 경보시스템 | 150원 | SVG 경광등 점멸, 구조대상 이동 속도 가속 |
| 🪜 탈출사다리 | 100원 | SVG 사다리 내려옴, 구조대상 즉시 탈출 |

### 4. 10초 침수 시뮬레이션

- SVG 지하/지상 단면도 — 물이 실시간으로 차오름
- 구조대상 캐릭터가 SVG 안에서 이동 → 사다리 타고 탈출
- 수위 게이지: 0~29cm 🟢 / 30~59cm 🟠 / 60cm+ 🔴
- 타이머 10초 미만 → 빨간 깜빡임
- 스킬 사용 시 → "+시간을 벌었어요!" 토스트

### 5. 태풍 예보 변화 이벤트 (2·3단계 전용)

플레이 시작 후 **10초 중 특정 시점**에 예보 업데이트 팝업:

> "태풍은 바다 쪽으로 지나갈 것 같아요." → "태풍이 도시 쪽으로 가까워졌어요! ⚠️"

| 예보 변화 | 게임 영향 |
|---------|---------|
| 태풍 가까워짐 | 강수량 증가, 침수 속도 빨라짐 |
| 경로가 도시로 이동 | 하천범람 이벤트 (+수위 15cm 즉시) |

- **2단계**: 바람 강함 → 경보시스템 1초 지연
- **3단계**: 하천범람 → "🚨 하천이 넘쳤어요!" + 수위 급상승

### 6. 등급 시스템

| 등급 | 조건 |
|------|------|
| S 🏆 | 선택한 3명 전원 탈출 |
| A 🎉 | 필수 대상 + 선택 1명 탈출 |
| B 👍 | 필수 대상(선생님+친구들) 탈출 |
| C 😰 | 필수 대상 1명 실패 → 패널티 |
| F 😢 | 필수 대상 전원 실패 → 패널티 |

3번 연속 실패 → 긴급 복습 패널 (안전 수칙 3개 체크 후 재도전)

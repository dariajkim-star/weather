# 🌊 Flood Escape Lab

> 초등학생 대상 지하 침수 탈출 체험 학습게임

기상청 실제 집중호우 데이터를 기반으로, 초등 1~2학년이 방재 장비를 선택하고 구조대상을 탈출시키는 28초 침수 시뮬레이션 게임입니다.

**공모전**: 기상청·기후에너지환경부 주관 「기상·기후 AI 해커톤 2026」 출품작

---

## 🎮 게임 화면 흐름

```
로비 → 구조대상 선택 → 장비 선택 → 플레이 → 결과(S/A/B) or 패널티(C/F)
```

---

## ✨ 핵심 기능

### 1. 시나리오 구성 (5단계)

| 단계 | 유형 | 설명 | 강우량/강도 | 난이도 | 해금 조건 |
|------|------|------|----------|-------|---------|
| ⭐ 1단계 | 집중호우 A | 2020 부산 초량 침수 | 80mm/h | 쉬움 | 시작부터 |
| ⭐⭐ 2단계 | 집중호우 A | 2023 오송 지하차도 | 90mm/h | 보통 | 1단계 B+ |
| ⭐⭐⭐ 3단계 | 집중호우 A | 2022 강남 집중호우 | 141mm/h | 어려움 | 2단계 B+ |
| 🌀 4단계 | 태풍 근접 B | 태풍이 도시 근처를 지나가요 | 강함 | 보통 | 3단계 B+ |
| 🌀🌀 5단계 | 태풍 직접 C | 태풍이 도시를 바로 지나가요 | 매우 강함 | 어려움 | 4단계 B+ |

B등급 이상 달성 시 다음 단계 해금. 데이터 출처: 기상청 ASOS API.

### 2. 구조대상 선택 (6명 중 3명)

👩‍🏫 선생님 · 🧒 우리 반 친구들 · 🐶 반려견 · 👨‍👩‍👧‍👦 우리가족 · 🎀 티니핑 · 🎤 아이브

### 3. 방재 장비 구매 (예산 500원)

| 장비 | 가격 | 플레이 이펙트 |
|------|------|------------|
| 🚧 물막이판 | 200원 | SVG 입구 장벽 등장, 수위 상승 40% 감소 (8초) |
| ⚙️ 배수펌프 | 250원 | SVG 하단 펌프, 수위 10cm 즉시 감소 |
| 🚨 경보시스템 | 150원 | SVG 경광등 점멸, 구조대상 이동 속도 가속 |
| 🪜 탈출사다리 | 100원 | SVG 사다리 내려오는 애니메이션, 구조대상 즉시 탈출 |

### 4. 28초 침수 시뮬레이션

- SVG 지하/지상 단면도 — 물이 실시간으로 차오름
- 구조대상 캐릭터가 SVG 안에서 실제로 이동, 사다리를 타고 탈출
- 수위 게이지: 0~29cm 🟢 / 30~59cm 🟠 / 60cm+ 🔴
- 타이머: 10초 미만 → 빨간 깜빡임 / 스킬 사용 시 "+시간을 벌었어요!" 토스트

### 5. 태풍 예보 변화 이벤트 (시나리오 B·C 전용)

플레이 시작 후 **10초 시점**에 예보 업데이트 발생:

> "태풍은 바다 쪽으로 지나갈 것 같아요." → "태풍이 도시 쪽으로 가까워졌어요! ⚠️"

| 예보 변화 | 게임 영향 |
|---------|---------|
| 태풍이 멀어짐 | 강수량 감소, 침수 속도 느려짐 |
| 태풍이 가까워짐 | 강수량 증가, 침수 속도 빨라짐 |
| 경로가 도시로 이동 | 하천범람 이벤트 (+수위 15cm 즉시) |
| 이동 속도 빠름 | 준비 시간 체감 감소 |
| 이동 속도 느림 | 비 오래 내려 누적 침수 증가 |

**시나리오 B**: 바람 강함 → 경보시스템 1초 지연 효과
**시나리오 C**: 하천범람 발생 → "🚨 하천이 넘쳤어요!" 배너 + 급격한 수위 상승

### 5. 등급 & 학습 목표

| 등급 | 조건 |
|------|------|
| S 🏆 | 선택한 3명 전원 탈출 |
| A 🎉 | 필수 대상 + 선택 1명 탈출 |
| B 👍 | 필수 대상(선생님+친구들) 탈출 |
| C 😰 | 필수 대상 1명 실패 → 패널티 화면 |
| F 😢 | 필수 대상 전원 실패 → 패널티 화면 |

성공 시 학습 목표 재노출 + 안전 약속 표시.
3번 연속 실패 시 긴급 복습 패널 (안전 수칙 3개 체크 후 재도전).

태풍 시나리오 학습 포인트:
- 태풍이 가까이 오면 비와 바람이 함께 강해져요
- 예보를 계속 확인해야 하는 이유를 게임으로 체험

## 🎨 디자인 원칙

- 배경 `#EFF6FF` (하늘색) · 카드 `#FFFFFF` · 텍스트 `#1E3A5F`
- 노란 우비 캐릭터 + 지하/지상 단면도 일러스트
- **긴박감은 색상이 아닌 애니메이션·수위 게이지·효과음으로 표현** — 초등 1~2학년이 공포감 없이 몰입 가능

## 🛠 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | React + TypeScript + Vite |
